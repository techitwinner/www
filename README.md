# www

This is the source code for my personal website, hosted at yourwebsite.com. The project is built exclusively with Bun, so be sure to have Bun installed if you want to run it locally.
Getting Started

To use this repository as a template for your own site, follow these steps:

1. **Clone the repository**
```bash
git clone https://github.com/yourusername/www.git
cd www
```

2. **Install Dependencies Use Bun to install the required dependencies:**
```bash
bun install
```

3. **Run Locally To start the development server, run:**
```bash
bun run dev
# or
bunx --bun astro dev
```

4. **Replace my data**
Update any personal data (such as metadata, content, and links) with your own in the relevant files.

**Enjoy coding!**
